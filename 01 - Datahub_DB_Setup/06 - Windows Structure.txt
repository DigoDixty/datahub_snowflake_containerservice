================================================================================================================================
PREPARE IMAGES FROM DATAHUB - DOCKER COMPOSE
================================================================================================================================
Go to datahub quickstart if you want to get docker compose or some more information:
https://docs.datahub.com/docs/quickstart

Install Docker Desktop
create a folder
cd \
mkdir datahub
create a file to yml of docker-compose
https://raw.githubusercontent.com/datahub-project/datahub/master/docker/quickstart/docker-compose-without-neo4j-m1.quickstart.yml
create a file compose.yml
on the same directory that saved file, run command:
docker compose up
after get all images
docker compose stop
